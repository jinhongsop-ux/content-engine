# Delivery Scope

## Included

- Local Node.js + Express application.
- Current UI workbench.
- Current backend routes and generation workflow.
- Default templates.
- Fictional demo site data.
- Documentation.
- Smoke test.
- Release zip builder.

## Not Included

- Mature SaaS hosting.
- Multi-user login or permission management.
- Payment or billing.
- Automatic production publishing without human review.
- SEO, traffic, inquiry, or revenue guarantees.
- Model, API, WordPress, hosting, or server fees.
- Real customer data in the release package.
