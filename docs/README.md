# Content Engine Lite v0.1

Content Engine Lite is a local, internal AI content production workbench. It helps organize site data, keyword tasks, project instructions, prompt strategy, outline generation, article generation, QA checks, image planning, publish packs, and WordPress draft publishing.

It is not a mature SaaS product. It is not an unattended publishing system. Generated content must be reviewed by a human before use.

## Best Fit

- Internal content operations.
- Multi-site blog planning.
- Semi-automated article drafting.
- QA-assisted editorial review.
- Preparing HTML, SEO metadata, image planning, and WordPress draft posts.

## Not Included

- SEO ranking guarantees.
- Traffic, inquiry, or sales guarantees.
- Multi-user permission systems.
- Payment, billing, or SaaS tenant management.
- Fully automated publish-without-review workflow.

## Main Modules

- `内容任务`: keyword task table, filters, selection, outline generation, article generation, WordPress publishing.
- `文章工作台`: outline, article preview, HTML source, QA, publish pack, images, data pack.
- `站点资料库`: brand settings, author profile, category knowledge, links, style reference, Project instructions.
- `导入中心`: file upload, AI recognition, field mapping, write results, site isolation.
- `设置`: article model, Gemini helper model, WordPress, default strategy entry.

## Human Review Required

Every generated article, SEO title, meta description, image ALT, schema, and WordPress draft should be reviewed before publishing.
