# B2B Site Template

这是一套适配 Content Engine 的 B端/外贸/制造站上传资料模板。

使用方式：

1. 复制整个 `b2b-site-template` 文件夹。
2. 按真实网站数据修改 `site.json`、`author.json`、`knowledge.json`、`links.json`、`style-reference.json`、`project-instructions.md` 和 `keywords.csv`。
3. 不要填写虚假的认证、工厂规模、MOQ、交期、客户案例或测试数据。
4. 在程序里创建 B端站点后，把这些文件压缩成 ZIP 上传到「导入中心」。

必需文件：

- `site.json`：公司/网站基础变量。
- `author.json`：团队/公司可信度档案。
- `knowledge.json`：产品、工艺、参数、认证、案例、采购 FAQ。
- `links.json`：产品页、解决方案页、案例页、认证页、联系/RFQ 页。
- `style-reference.json`：文章 HTML 风格参考。
- `project-instructions.md`：站点级最高优先级写作指令。
- `keywords.csv`：B端关键词任务表。
