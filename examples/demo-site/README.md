# Demo Site

`examples/demo-site/` contains fictional data for demonstrating Content Engine Lite v0.1.

It does not use real customer data, real brands, real domains, API keys, or WordPress credentials.

Files included:

- `site.json`
- `author.json`
- `knowledge.json`
- `links.json`
- `style-reference.json`
- `project-instructions.md`
- `keywords.csv`
